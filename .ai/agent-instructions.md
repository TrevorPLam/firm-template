# AI Agent Instructions - Firm Template

## Repository Overview
This is a Next.js template for professional services firms, including both frontend and backend components.

## Architecture
- **Frontend**: Next.js 14+ with App Router
- **Backend**: Python/Django backend service
- **Features**: Blog, Contact, Scheduling, Video, Theme Editor

## Key Conventions
- Frontend uses Next.js App Router
- Backend follows modular structure in `backend/modules/`
- Features are organized in `features/` directory
- Shared UI components in `components/ui/`

## Common Tasks
- **Frontend page**: Create in `app/[route]/page.tsx`
- **Backend module**: Create in `backend/modules/[module-name]/`
- **Feature**: Create in `features/[feature-name]/` with components/ and lib/
