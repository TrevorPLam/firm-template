# API Gateway Service

This is the Django backend API service.

## Framework
- **Django REST Framework**
- **Python**

## Structure
- `backend/api/` - API endpoints organized by feature
- `backend/modules/` - Domain modules with business logic
- `backend/config/` - Django settings and configuration

## Architecture
- Follows Clean Architecture principles
- Domain modules in `modules/`
- API endpoints in `api/`
- Shared utilities in `lib/`

## Development
- Run migrations: `python manage.py migrate`
- Create superuser: `python manage.py createsuperuser`
- Run server: `python manage.py runserver`
