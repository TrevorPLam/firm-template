# Onboarding Guide

Follow this checklist to get set up and productive quickly.

## Day 1 setup

```bash
# Clone the repository and install dependencies.
git clone git@github.com:firm/firm-template.git
cd firm-template
pnpm install
```

## Environment checklist

- [ ] Access to secrets manager and CI dashboards
- [ ] Local environment variables configured
- [ ] Verified access to staging deployments

## First tasks

1. Read [Platform Guidelines](../PLATFORM.md).
2. Review [Documentation Index](../README.md).
3. Complete the training exercises in [Team Training Curriculum](../training/README.md).
