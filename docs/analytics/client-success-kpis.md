# Client Success KPIs

## KPI Definitions

| KPI | Definition | Owner |
| --- | --- | --- |
| Retention | % clients renewed | Account lead |
| Adoption | % features used | Success lead |
| Satisfaction | CSAT score | Success lead |

## KPI Calculation Example

```ts
const retentionRate = (renewedClients / totalClients) * 100;
const adoptionRate = (activeFeatures / totalFeatures) * 100;
// Track KPIs monthly for trend analysis.
```

## Reporting Cadence

- Weekly KPI review for portfolio owners.
- Monthly executive summary.
