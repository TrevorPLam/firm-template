# Financial Performance Tracking

## Financial Metrics

- Monthly recurring revenue (MRR)
- Gross margin by client
- ROI per engagement

## ROI Example

```ts
const roi = ((benefit - cost) / cost) * 100;
// Capture ROI quarterly to align with client renewals.
```

## Review Cadence

- Monthly finance review with leadership.
- Quarterly adjustments to pricing strategy.
