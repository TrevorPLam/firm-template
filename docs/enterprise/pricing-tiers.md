# Enterprise Pricing Tiers & Models

## Tier Definitions

| Tier | Target | Key Inclusions |
| --- | --- | --- |
| Growth | 5-20 clients | Core platform + onboarding |
| Scale | 20-100 clients | Automation + analytics |
| Enterprise | 100+ clients | Dedicated support + governance |

## Value Metric Examples

- Sites launched per quarter
- Portfolios managed
- Governance features enabled

## Packaging Notes

- Anchor pricing to outcomes (time saved, throughput).
- Offer annual commitments with implementation support.
