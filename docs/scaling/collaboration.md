# Collaboration Framework

## Communication Cadence

- Daily async standup (Slack updates).
- Weekly portfolio review.
- Monthly retro for process improvements.

## Documentation Standards

```md
## Release Summary
- Scope:
- Risks:
- Rollback plan:
```

> Keep summaries within 10 lines to maintain scannability.

## Feedback Loop

- Capture blockers in a shared risk register.
- Highlight systemic issues in monthly retros.
